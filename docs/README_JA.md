# NB Command

[EN](../README.md) | [中文](README_CN.md) | [日本語](README_JA.md) | [한국어](README_KO.md)

## 概要

NB Commandは[Nebula](https://github.com/Melledy/Nebula)用に特別に設計されたグラフィカルなリモートコマンド実行ツールです。

## 特徴

- 多言語対応：中国語、英語、日本語、韓国語をサポート
- ゲーム管理：プレイヤー管理、アイテム管理、キャラクター管理などの機能を提供
- グラフィカルインターフェース：直感的な操作画面で複雑なコマンドの実行プロセスを簡素化
- 設定の保存：サーバーアドレスと認証トークンを自動的に保存
- コマンド履歴：実行したコマンド履歴を記録し、追跡と再利用を容易にします
- データハンドブック：組み込みのキャラクターやアイテムデータハンドブックで、すばやく検索・選択が可能

## インストールと実行

1. 最新のリリースからお使いのプラットフォームに対応する実行可能パッケージをダウンロードしてください [releases](https://github.com/HongchengQ/NB-Command/releases)
2. 展開して実行ファイルを実行してください
> どのnbcommand-windows.zipをダウンロードすべきか分からない場合は、これが最適な選択です

またはソースからビルド：
- Java 21
- maven

```bash
# アプリケーションの実行
mvn javafx:run
```

```bash
# 実行可能ファイルのビルド
mvn package
```
出力ディレクトリは`/target/nbcommand`にあります

## 使用方法

1. 上部にサーバーアドレスと認証トークンを入力してください
2. 左側のカテゴリリストからコマンドカテゴリを選択してください
3. 中央のリストから特定のコマンドを選択してください
4. 必要に応じてコマンドパラメータを入力してください
5. 生成されたコマンドをプレビューし、正しいことを確認した後に実行してください

### トークンの取得方法
> Nebula > `config.json` > `remoteCommand` > `useRemoteServices` が `true` であることを確認してください

#### 管理者権限
> Nebula > `config.json` > `remoteCommand` > `serverAdminKey` をトークンとして使用

#### ユーザー権限
> ゲーム内でコマンド `!remote` を使用すると、画面にトークンが表示されます
